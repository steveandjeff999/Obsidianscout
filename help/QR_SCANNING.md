# QR Scanning Guide

The QR scanning feature allows users to quickly import and export scouting data using QR codes. This is especially useful for offline data transfer between devices at events.

## How to Use QR Scanning

1. **Navigate to the QR Scanning Page:**
   - Go to the Scouting section and select the QR Scan option.
2. **Scan a QR Code:**
   - Use your device's camera to scan a QR code generated by another device.
   - Make sure your browser has camera permissions enabled.
3. **Export Data as QR:**
   - On the data export page, select the option to generate a QR code.
   - Show this code to another device for scanning.

## Troubleshooting

- **Camera Not Detected:**
  - Ensure your browser has permission to access the camera.
  - HTTPS is required for camera access in most browsers.
- **QR Code Not Recognized:**
  - Make sure the QR code is fully visible and not damaged.
  - Clean your camera lens and try again.

## Tips
- Use a well-lit environment for scanning.
- If scanning fails, try refreshing the page or restarting your browser.
